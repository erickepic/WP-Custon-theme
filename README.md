# Gtheme - WP Development Theme

**Gtheme** is a modern, high-performance WordPress boilerplate designed for professional developers. 

## Identity
- **Name**: Gtheme
- **Focus**: Build. Code. Design.
- **Framework**: Integrated with `gframe` for ACF blocks, custom functions, and shortcodes.

## Features
- **Clean Structure**: Minimal boilerplate files following WordPress best practices.
- **Developer First**: Truncated `functions.php` for better maintainability.
- **Modular Framework**: `gframe` based logic separation.
- **Optimized Assets**: Modern CSS organization and structure.

## Folder Structure
- `/gframe`: Core framework logic (ACF Blocks, Styles, Functions, Shortcodes).
- `/css`: Standardized CSS files.
- `/js`: Custom JavaScript modules.
- `/assets`: Images, Fonts, and static assets.
- `/page-templates`: Custom WordPress page templates.

*Created and maintained as a clean starting point for Gtheme projects.*
